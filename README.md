> A centralized, web-based onboarding ecosystem designed to streamline knowledge transfer for the Strategic Global Development (SGD) department at AIESEC in Universitas Muhammadiyah Malang.

## 📌 Executive Summary
Effective leadership requires efficient systems. As the **Team Leader of Strategic Global Development (SGD)**, I identified a bottleneck in how new members absorbed departmental knowledge. To solve this, I developed this lightweight onboarding portal. It serves as a single source of truth, combining my technical background in Informatics with my organizational management responsibilities.

## 🚨 The Problem
Previously, functional onboarding materials were scattered across various platforms (PDFs, Google Drive folders, and chat histories). This fragmented approach led to:
- Slower adaptation time for new members.
- Information loss and inconsistency.
- High dependency on one-on-one manual explanations.

## 💡 The Solution
I engineered this static web portal to centralize all learning modules, operational guidelines, and essential resources. By applying **human-centered design**, the interface is built to be intuitive, ensuring that members can independently navigate their learning journey.

## ✨ Key Features
- **📚 Centralized Modules:** All functional knowledge, from basic introductions to advanced operational standard procedures, in one accessible place.
- **⚡ Lightweight & Fast:** Built strictly with HTML5 and CSS3 to ensure zero loading lag and maximum accessibility, even on slow network connections.
- **📂 Resource Hub:** Direct, embedded links to essential organizational tools, trackers, and external documents.
- **📱 Responsive Layout:** Designed to be easily readable on both desktop and mobile devices for on-the-go learning.

## 🛠️ Tech Stack
- **Markup:** HTML5 (Semantic HTML for better structure and accessibility)
- **Styling:** CSS3 (Custom styling, Flexbox/Grid for layout management)
- **Version Control:** Git & GitHub
